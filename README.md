# PassGen

Simple static password generator.

Features:
- Adjustable length & count
- Character classes: Uppercase, Lowercase, Numbers, Symbols (editable & sanitized), optional Whitespace
- Constraints: Begin with letter, No similar characters, No duplicates, No sequential characters
- Save preferences (cookie)
- Batch generation with constraint retries
- Copy each password with one click (gutter icon)
- Collapsible sections with disclosure triangles

Version: v0.0.3
