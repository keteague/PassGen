# PassGen

Simple static password generator.

Features:
- Adjustable length & count (fixed or variable range)
- Character classes: Uppercase, Lowercase, Numbers, Symbols (editable & sanitized), optional Whitespace
- Constraints: Begin with letter, No similar, No duplicates, No sequential
- Minimum symbol quota (1-9, enabled by default)
- Affixes: Prefix & Suffix (mirror, custom) with length integration
- Per-password length badge
- Save preferences (cookie + localStorage fallback)
- Batch generation with constraint retries & specific failure reasons
- Per-password selection checkboxes
- Copy All / Select All / Download selected passwords
- Copy each password with one click (gutter icon)
- Reset Defaults modal to restore baseline settings
- Collapsible sections with disclosure triangles
