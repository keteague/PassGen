Password Generator v0.03.33

    API and URL parameters

    Use query parameters to generate passwords via a crafted URL. Append parameters to this page’s URL. If format=text is provided, the page renders plaintext suitable for curl.

    Common parameters:

    • mode: password | passphrase | hex
    • length (with lengthMode=fixed) or min/max (implies range)
    • count: number of passwords
    • format: text (outputs plaintext)

    Password options:

    • optUpper, optLower, optNums, optSpace, optBeginLetter, optNoSimilar, optNoDuplicates, optNoSequential = 1|0
    • symbols: custom symbols string
    • forceSymbols=1|0, forceCount=1..9

    Affix options:

    • optPrefix=1 to enable; prefixType=uppercase|lowercase|numbers|symbols|custom; if not custom use prefixLen=1..32, else prefixCustom=string
    • optSuffix=1 to enable; suffixType=uppercase|lowercase|numbers|symbols|custom|mirror; if not custom or mirror use suffixLen=1..32, else suffixCustom=string

    Passphrase options:

    • ppWords=3..10, ppDelimiter, ppCasing, ppIncludeNumber=1|0, ppWhitespace=1|0, ppWhitespaceRestrict=1|0, ppWhitespaceRestrictMode, ppReplacements=1|0

    Hex options:

    • hexCasing=upper|lower, hexBeginLetter=1|0, hexNoSequential=1|0

    Examples:

    	# Password, 16 chars, require symbol quota, plaintext
	?mode=password&length=16&forceSymbols=1&forceCount=2&format=text

	# Password, range 12-18, 50 results
	?mode=password&min=12&max=18&count=50

	# Passphrase: 4 words, dash delimiter, upper casing
	?mode=passphrase&ppWords=4&ppDelimiter=-&ppCasing=upper

	# Passphrase, range 20-30 exact length fitting
	?mode=passphrase&min=20&max=30&ppWords=3

	# Hex: uppercase, begin with letter, 32 length
	?mode=hex&length=32&hexCasing=upper&hexBeginLetter=1

	# Hex: range 24-48, lowercase, no sequential, plaintext
	?mode=hex&min=24&max=48&hexCasing=lower&hexNoSequential=1&format=text

	# Prefix/Suffix: 2-letter uppercase prefix, mirrored suffix
	?mode=password&length=20&optPrefix=1&prefixType=uppercase&prefixLen=2&optSuffix=1&suffixType=mirror

	# curl usage (plaintext)
	curl -s "https://example.com/index.html?mode=password&length=20&format=text"

    Tip: Use the “Copy URL” button in the PASSWORDS section to copy a crafted URL matching your current settings. Append &format=text for plaintext output suitable for curl.
